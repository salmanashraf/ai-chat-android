# Getting Started

This guide walks you through integrating the AI Chat library into your Android app.

## Requirements
- Android 8.0+ (API 26+)
- Kotlin 1.9+
- Jetpack Compose recommended
- Coroutine support

## Installation

Add the dependency (coming soon on Maven Central):

```gradle
implementation("com.salmanashraf.ai:core:1.0.0")
implementation("com.salmanashraf.ai:ui:1.0.0")
